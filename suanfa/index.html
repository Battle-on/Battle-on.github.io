杨辉三角

var generate = function(numRows) {
    if(numRows===1){
        return [[1]]
    }
    const arr = []

    for(let i=0;i<numRows;i++){
        arr[i] = []
            for(let j=0;j<=i;j++){
                arr[i][j] = []
                if(j==0 || j==i) {
                    arr[i][j]=1;    //边界赋值为1   
                }else{
                    arr[i][j]=arr[i-1][j-1]+arr[i-1][j];
                }
            }
    }
    return arr
};

递归的高阶应用-memorization

经典案例：爬楼梯
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2个台阶。你有多少种不同的方法可以爬到楼顶呢？注意：给定 n 是一个正整数

我们，分析一下，我们找下规律，因为题目限定了 值可以爬 1 或者2个台阶，也就是说，到底最高层台阶的前一步只有两种选择，要不就是 在 第 n - 1
要不就是 在 第 n - 2层

一层楼梯： 1
两层楼梯： 1 + 1， 2
三层楼梯： 1 + 2，1+1+1 ， 2+1
四层楼梯： 1+1+2 2+2 1+2+1 1+1+1+1 2+1+1

根据需求，代码如下：

const map = new Map()
const climbStairs = function(n) {
  let reslut = ''
  if(map.get(n)){
    return map.get(n)
  }
  if(n <= 3) {
      reslut =  n
  }else {
    reslut =  climbStairs(n - 1) +  climbStairs(n-2)
  }
  map.set(n, reslut)
  return reslut
};
